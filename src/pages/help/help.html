Quick Start (60 seconds)

  1. 1. Install the Code Bridge extension.
  2. 2. Open the popup and connect GitHub (optional, for uploads).
  3. 3. Open a supported platform (Codeforces, LeetCode, HackerRank) and start a problem.
  4. 4. Solve as normal; the timer tracks you automatically.
  5. 5. When you get Accepted, the session ends and your solution is saved.

How Code Bridge Works (Conceptually)

  1. You open a problem and start working.
  2. The timer tracks your work per problem on each platform, even if you navigate between pages.
  3. When you get Accepted, the session ends automatically.
  4. Your solution and metadata are saved locally in your browser.
  5. If Auto-save is on, Code Bridge uploads to GitHub for you.

GitHub Sync (Optional)

  1. 1. Click "Authenticate with GitHub" in the popup.
  2. 2. Copy the code shown and open the GitHub link.
  3. 3. Paste the code on GitHub's verification page.
  4. 4. Once authorized, the workflow panel appears in the popup.

Customized Experience

Customize commit messages, file paths, and README formats with variables like [id] or [difficulty].

Auto-Save Mode

Enable Auto-save in Settings to push your solution after an Accepted submission.
