How Code Bridge Works (Conceptually)

  • 1. You solve a problem.
  • 2. The timer runs in the background.
  • 3. When you get Accepted, the session ends.
  • 4. Your solution + metadata get saved locally.
  • 5. (Optional) Auto-save pushes it to GitHub.

The timer automatically tracks your work per problem on each platform ‚Äî even if you navigate between pages.

Getting Started (GitHub Auth)

  • 1. Click "Authenticate with GitHub" in the popup.
  • 2. Copy the code shown and click the GitHub link.
  • 3. Paste the code on GitHub's verification page.
  • 4. Once authorized, Code Bridge will show the workflow panel.
