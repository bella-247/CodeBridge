Getting Started (GitHub Auth)

  • 1. Click "Authenticate with GitHub" in the popup.
  • 2. A code will appear. Copy it and click the link to GitHub.
  • 3. Paste the code on GitHub's verification page.
  • 4. Once authorized, CodeBridge will reveal the workflow panel.

Customized Experience

Use the Customize link to define your own commit messages, file paths, and README formats using variables like [id] or [difficulty].

Auto-Save Mode

Enable "Auto-save" in Settings to automatically push your solution whenever the platform signals an Accepted submission.
